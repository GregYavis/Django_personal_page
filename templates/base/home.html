Пара небольших программ, взаимодействующих с Instagram API и VK API – одни из первых проектов, в которых я начал использовать объектно-ориентированное программирование (https://github.com/GregYavis/Instagram_posting, https://github.com/GregYavis/LikesDeleter_with_VKAPI).
